## 📌 Project Overview

This project analyzes customer shopping behavior using a dataset of ~3,900 transactions across multiple product categories.

The goal is to extract actionable insights into customer purchasing patterns, discount usage, and subscription behavior to support **data-driven business decisions**.

---

## 🚀 Live Highlights

* 📊 Built an interactive **Power BI dashboard**
* 🧠 Performed **end-to-end data analysis** (Python → SQL → Visualization)
* 🎯 Generated **business-ready insights & recommendations**

---

## 🎯 Objectives

* Analyze customer demographics and behavior
* Identify purchasing trends and preferences
* Evaluate discount and promotion effectiveness
* Segment customers (New, Returning, Loyal)
* Assess subscription impact on revenue

---

## 📂 Dataset Summary

* **~3,900 transactions**
* **18 features**
* Multi-category retail dataset

### Key Features:

* 👤 Demographics: Age, Gender, Location
* 🛒 Purchases: Category, Item, Amount
* 📈 Behavior: Discounts, Ratings, Frequency, Shipping

---

## 🧹 Data Processing (Python)

* Data cleaning and validation using **Pandas**
* Missing values handled via **median imputation**
* Standardized column names (`snake_case`)
* Feature engineering:

  * `age_group`
  * `purchase_frequency_days`
* Removed redundant columns to improve efficiency

---

## 🗄️ SQL Analysis (PostgreSQL)

### Key Business Questions:

* 💰 Revenue by gender
* 🎯 High-spending discount users
* ⭐ Top-rated products
* 🚚 Shipping type comparison
* 🔁 Subscription vs non-subscription behavior

---

## 📊 Key Insights

### 💰 Revenue Insights

* Male customers generated higher total revenue
* Discount users can still be high-value customers
* Express shipping correlates with slightly higher spending

### 📦 Product Insights

* Top-rated products: Gloves, Sandals, Boots
* Some products rely heavily on discounts
* Product performance varies across categories

### 👥 Customer Insights

* Loyal customers dominate the dataset
* Returning customers show strong engagement
* Low proportion of new customers

### 🔔 Subscription Insights

* Similar average spend between subscribers & non-subscribers
* Non-subscribers contribute higher total revenue

### 🎯 Demographic Insights

* Young adults generate the highest revenue
* Revenue distribution is fairly balanced across age groups
